Cyclic Voltammetry analysis

Caution

This page is still being edited and reviewed.

1. Supporting file formats​

For Cyclic Voltammetry analysis, we are now only support JCAMP-DX files (.jcamp, .dx, *.jdx), zip (in BagIt format) files.

2. Upload your data and view it​

To upload and view your data, you can follow this instruction

3. Working with your data​

  1. Select which spectrum that you want to view

  2. Select which spectrum that you want to work with

  3. Select to add max and min peaks

  4. Select to add peckers

4. Download your analysed data as table format​

  1. Open analyses tab, scroll down to Datasets section, click on which data you want to see the details

  2. Select your analysed csv file to download
