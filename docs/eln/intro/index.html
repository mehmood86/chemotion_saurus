How to Use This Manual

Hello and thank you for using Chemotion ELN!

If you are a first-time user, we recommend that you read the manual before you start working on your project, since it's not always possible to make the functionality of Chemotion ELN self-explanatory. In this manual, we will highlight what's necessary to make full use of the ELN's capabilities.

If you have specific questions, please check out the links below. They correspond to the chapters of the manual:

  • If you do not know where to start at all, go here.

  • Find information on the general background of the Chemotion ELN here.

  • If you need support on how to report and structure your data, you can find videos here.

  • If you need help accessing the Chemotion ELN, follow this link.

  • Please have a look at the definition of terms that we'll use throughout the manual.

  • You can learn how to adjust your user interface here.

  • In the ELN, projects are organized in collections.

  • Please also have a look at using lists.

  • In the Chemotion ELN, you will work with elements, such as samples, reactions and wellplates.

  • Detail modals provide information about your structures, reactions and wellplates.

  • Learn about toolbar functions here.

  • Questions about how to enter molecules into the Chemotions ELN will be answered in this chapter.

  • With Chemotion ELN, you can scan and generate QR codes as well as barcodes.

  • If you want to know how to create a group-inventory, follow this link.

  • The access of analytical data via the Chemotion ELN is explained here.

  • Moreover, you can customize the ELN using generic elements and segments.

  • Here, you can find solutions for common issues.

While reading the manual, please mind the hints and instructions that are identified by the following symbols for Caution and Recommendation.

Caution

Very important facts you need to consider for a correct use of the Chemotion ELN.

Recommendation

Important hints.

info

Additional remarks and recommendations.

Links

Links that explain external tools and use of the Chemotion ELN.

All terms in this manual that are written in boldface correspond to terms that are used in the user interface.

If your questions are not answered in the manual, please contact us! The ELN team is available for questions and assistance, but we do assume that you are familiar with this manual and, in particular, with the highlighted Cautions and Recommendations.
